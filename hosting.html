Frequently asked questions

What infrastructure do you use?

We deploy on reliable cloud infrastructure with PostgreSQL databases, automated backups, and SSL certificates. The exact stack may vary by plan but always meets our performance and reliability standards.

Can I use my own domain?

Yes. All plans include custom domain support. Just point your DNS to our servers and we'll handle SSL certificate provisioning automatically.

Can I migrate an existing site?

Absolutely. If you already have a Panda CMS site running elsewhere, we can migrate it to our managed hosting. We'll handle the database migration, file transfer, and DNS switchover.

What if I need to scale?

You can upgrade your plan at any time. If you're on the Professional plan and need more resources, we can move you to a custom Enterprise configuration. No downtime required.

Do I need a Pro license separately?

The Professional and Enterprise hosting plans include a Pro license at no extra cost. On the Starter plan, you can add a Pro license separately for £120/yr if needed.

Ready to get started?

Tell us about your project and we'll set up your managed Panda CMS instance. No commitment, cancel any time.
