Build on panda-core

Want to build your own Rails application with OAuth authentication already sorted? Use panda-core as your foundation. You get user management, multi-provider OAuth, and a clean configuration system out of the box. 

# Gemfile
gem "panda-core"

# Then configure your OAuth providers
Panda::Core.configure do |config|
  config.authentication_providers = {
    google: { client_id: ENV["GOOGLE_ID"], ... }
  }
end
