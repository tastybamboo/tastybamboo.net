Panda CMS

Better websites, on Rails

A modern, modular content management system built for Ruby on Rails. Simple enough for small sites, powerful enough to scale. No bloat, just the features you need.

bundle add panda-cms

Up and running in minutes

Three commands to a working CMS. No complex configuration required.

1

Install

bundle add panda-cms

Add the gem to your Rails app

2

Setup

rails g panda:cms:install rails db:migrate

Run the generator and migrations

3

Launch

rails server

Visit /admin to start editing

Everything you need to manage content

A complete content management toolkit that integrates seamlessly with your Rails application. Built by developers, for developers.

Hierarchical Pages

Unlimited nesting depth with automatic slug management, redirects when paths change, and multiple status states.

Blog & Posts

Date-based URLs, author tracking, excerpts, and full SEO support. Perfect for news, blogs, and announcements.

Page Templates

Define reusable layouts with block-based content composition. Create consistent page structures across your site.

Flexible Menus

Auto-generated from page hierarchies or manually managed. Menus update automatically when pages change.

Form Builder

Built-in spam protection with timing detection, honeypots, and rate limiting. WCAG-compliant, no CAPTCHAs needed.

File Management

Gallery view for uploads with Active Storage integration. Manage images, documents, and media assets easily.

Import & Export

Bulk editor for JSON import/export of pages, posts, and menus. Migrate content easily between environments.

SEO built-in, not bolted on

Every page and post comes with comprehensive SEO controls. No plugins required.

  • SEO title and description with character limits
  • Open Graph metadata for social sharing
  • Canonical URLs and robots meta tags
  • SEO inheritance from parent pages
  • Automatic 301 redirects when URLs change 
# Every page includes:
seo_title:        "Custom SEO Title"
seo_description:  "Meta description..."
og_title:         "Social Share Title"
og_description:   "Social description..."
og_image:         "share-image.jpg"
canonical_url:    "https://..."
no_index:         false

A modern editor, not a bloated WYSIWYG

Panda Editor is built on EditorJS, delivering a clean, block-based editing experience. No TinyMCE bloat, just focused content creation.

Rich Block Types

Paragraphs, headers, lists, quotes, tables, images, alerts, code blocks, and delimiters. Everything you need.

Inline Footnotes

Add footnotes inline with automatic numbering and a collapsible sources section. Perfect for citations.

Import Anything

Built-in converters for HTML and Markdown. Migrate existing content or paste from anywhere. 

# config/initializers/panda.rb
Panda::Core.configure do |config|
  config.authentication_providers = {
    google: {
      client_id: ENV["GOOGLE_CLIENT_ID"],
      client_secret: ENV["GOOGLE_SECRET"],
      hd: "yourcompany.com"
    },
    github: {
      client_id: ENV["GITHUB_CLIENT_ID"],
      client_secret: ENV["GITHUB_SECRET"]
    }
  }
end

OAuth authentication, no passwords

Secure authentication via Google, Microsoft, or GitHub. Domain restrictions keep your admin locked down.

  • Multiple providers simultaneously
  • Domain restrictions (e.g., @yourcompany.com only)
  • First user auto-promoted to admin
  • Rake tasks for user management
